user_id는 GitHub actor.id이며 개인·bot·organization을 자동 구분하지 않습니다. 또한 “처음 관측” cohort는 전체 GitHub 생애 최초가 아니라 현재 fact 관측창에서의 최초입니다. lifecycle의 returning/unknown도 관측창 바깥 이력을 알 수 없어 진짜 신규 사용자와 구분할 수 없습니다.
